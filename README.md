# 🎲 Boardgames

Онлайн-платформа настольных игр с атмосферой реального игрового стола.
Первые игры: **Uno** и **Alias**. Реалтайм-мультиплеер, комнаты с друзьями и подбор случайных игроков.

## Принципы

- **Все изменения и планы — через GitHub Issues.** Каждая фича/баг — отдельный issue с метками.
- **Ощущение настоящей настолки** — карты в руке, стол, анимации раздачи и броска.
- **Mobile-first** — играть удобно с телефона с первого дня.
- Язык интерфейса: русский (архитектура готова к i18n).

## Структура монорепо

```
boardgames/
├── apps/
│   ├── web/        # фронтенд: React + Vite
│   └── server/     # бекенд: Node + Fastify + Socket.IO
├── packages/
│   └── shared/     # общие типы, события, правила игр (чистый TS)
├── docs/           # документация
└── design/         # дизайн-концепты
```
