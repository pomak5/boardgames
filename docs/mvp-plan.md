# План MVP

Каждый пункт = GitHub Issue. Метки: `infra`, `backend`, `frontend`, `game:codenames`, `game:uno`, `game:alias`, `design`, `bots`.

**Изменение приоритета (2026-06-11): первая игра — Коднеймс** (см. `docs/games/codenames.md`).
Порядок игр: Codenames → Uno → Alias. Актуальный статус — в трекер-issue с чекбоксами.

## Этап 0 — Фундамент (`infra`)

1. Скелет монорепо: npm workspaces, `apps/web`, `apps/server`, `packages/shared`, TS-конфиги, ESLint/Prettier
2. CI: GitHub Actions — lint + typecheck + tests на PR
3. Docker Compose: postgres + server + web (dev-режим)

## Этап 1 — Ядро платформы (`backend`, `frontend`)

4. Гостевая авторизация: ник → JWT, reconnect по токену
5. Комнаты: создать / войти по коду-ссылке, настройки (игра, кол-во игроков, боты), lifecycle
6. Socket.IO-слой + типизированные события в `shared`
7. Чат комнаты
8. Подбор случайных игроков (очередь + таймаут)
9. UI: главная (выбор игры), лобби комнаты, мобильная вёрстка

## Этап 2 — Коднеймс (`game:codenames`) — приоритет №1

22. Движок Codenames в `shared`: поле 5×5, ключ-карта, подсказки, ходы, победа/поражение (юнит-тесты)
23. Словарь слов RU (~400) + генерация поля
24. Бот-капитан: подсказки через word embeddings (navec), уровни риска
25. UI Codenames: поле, экран капитана, ввод подсказки, лог ходов, табло
26. Режим соло/дуо кооп: бот-капитан + «таймер-команда», настройки комнаты

## Этап 3 — Uno (`game:uno`)

10. Движок Uno в `shared`: колода 108, ходы, валидация, спецкарты (юнит-тесты)
11. Кнопка «UNO!» + штраф (+2 карты, окно челленджа)
12. Бот для Uno (простая эвристика)
13. UI стола Uno: рука веером, анимации раздачи/броска, выбор цвета, индикация хода
14. Завершение партии: подсчёт очков, экран результатов, rematch

## Этап 4 — Alias (`game:alias`)

15. Движок Alias в `shared`: команды, раунды, таймер, очки (угадано +1 / пропуск −1)
16. Словари по сложности: схема БД + сид (easy/medium/hard, ~500+ слов каждый)
17. UI Alias: экран ведущего (слово + «угадано/пропуск»), экран угадывающих, таймер, табло команд
18. Финал: победный счёт, экран результатов, rematch

## Этап 5 — Полировка

19. Применить выбранный дизайн-концепт ко всему UI (токены, компоненты)
20. Звуки карт и таймера (с выключателем)
21. Деплой на VPS: docker compose prod, домен, HTTPS

## Вне MVP (бэклог)

- Аккаунты с email, статистика, рейтинги
- Хаус-рулы Uno (стакание +2/+4, jump-in) как настройки комнаты
- Английский язык (i18n)
- Голосовой чат
- Новые игры (Дурак, Манчкин…)
