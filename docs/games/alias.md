# Alias — спецификация (MVP)

## Параметры комнаты

- Команды: 2+ (игроки распределяются в лобби, минимум 2 человека в команде)
- Длительность раунда: 60 сек (настраивается: 30/60/90)
- Победный счёт: 30 очков (настраивается)
- Словарь: по сложности — лёгкий / средний / сложный

## Ход игры

1. Команды ходят по очереди. В команде по кругу выбирается **ведущий** раунда.
2. Ведущий видит слово и объясняет его своей команде (нельзя однокоренные) — устно (созвон) или через чат комнаты.
3. **Подтверждает только ведущий**: кнопки «Угадано ✓» (+1) и «Пропуск ✗» (−1). Следующее слово показывается сразу.
4. Таймер истёк → раунд завершён, ход переходит следующей команде. Последнее слово сгорает (спорное правило «последнее слово для всех» — в бэклог).
5. Команда достигла победного счёта в конце полного круга → победа (при равенстве — дополнительный круг).

## Экраны

- **Ведущий:** крупное слово, таймер, кнопки ✓/✗, текущий счёт раунда
- **Команда ведущего:** таймер + счёт раунда (слово скрыто)
- **Соперники:** таймер + табло; видят слово ведущего (контроль честности) — отдельная настройка «показывать соперникам», по умолчанию вкл
- **Табло:** очки команд, кто ведёт, прогресс до победного счёта

## Словари

- Таблица `alias_words (word, difficulty)`; сид: ~500+ слов на уровень, русский язык
- Слова в партии не повторяются (трекинг использованных)

## Состояние (shared)

```ts
type AliasState = {
  teams: Team[]; // id, name, players, score
  currentTeam: number;
  explainer: PlayerId;
  round: {
    word: string;
    startedAt: number;
    duration: number;
    guessed: number;
    skipped: number;
  } | null;
  usedWordIds: Set<number>;
  targetScore: number;
  phase: 'lobby' | 'round' | 'between' | 'finished';
};
```

Действия: `startRound`, `markGuessed`, `markSkipped`, `endRound` (по таймеру сервера).
