# Uno — спецификация (MVP)

## Параметры комнаты

- Игроки: 2–10 (настраивается хозяином)
- Боты: вкл/выкл + количество (добивают до выбранного числа игроков)
- Правила: классические (хаус-рулы — в бэклоге как настройки)

## Колода (108 карт)

- 4 цвета × (один «0», по две «1–9», по две «Пропуск», «Разворот», «+2») = 100
- «Смена цвета» ×4, «+4 со сменой цвета» ×4

## Ход игры

1. Каждому раздаётся 7 карт, верхняя карта колоды открывается (если спецкарта — применяется по официальным правилам).
2. Ход: сыграть карту, совпадающую по **цвету или номиналу/символу**, либо любую чёрную; иначе — взять карту из колоды (взятую можно сразу сыграть, если подходит).
3. Спецкарты: Пропуск, Разворот (при 2 игроках = Пропуск), +2, Смена цвета, +4 (формально — только если нет карты нужного цвета; челлендж +4 — в бэклоге).
4. Кончилась колода → перетасовать сброс (кроме верхней).

## Кнопка «UNO!»

- При выкладывании предпоследней карты у игрока появляется кнопка «UNO!».
- Если не нажал до того, как любой соперник нажал «Поймать!» (до начала хода следующего игрока) — штраф **+2 карты**.
- Боты жмут UNO с задержкой 1–3 сек (чтобы было шанс поймать).

## Конец партии

- Игрок сбросил последнюю карту → партия окончена.
- Очки победителю с рук проигравших: цифры по номиналу, спецкарты 20, чёрные 50.
- Экран результатов → rematch тем же составом.

## Состояние (shared)

```ts
type UnoState = {
  deck: Card[];
  discard: Card[];
  hands: Record<PlayerId, Card[]>; // сервер шлёт каждому только его руку + счётчики чужих
  currentPlayer: PlayerId;
  direction: 1 | -1;
  activeColor: Color; // после чёрных карт
  pendingDraw: number; // накопленный +2/+4
  unoCallers: Set<PlayerId>;
  phase: 'playing' | 'choosingColor' | 'finished';
};
```

Действия: `playCard`, `drawCard`, `chooseColor`, `callUno`, `catchUno`.
